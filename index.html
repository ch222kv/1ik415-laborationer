Ibland kan man ju bli måttligt irriterad...

Jag satte mig ner och begrundade hur jag skulle kunna lösa uppgiften på bästa sätt. Centreringen och bredden var det första jag gjorde. Började uppifrån och arbetade mig nedåt. De mindre vitala delarna så som färger fick vänta till sist. När jag kände mig nöjd med varje delmoment gjorde jag en commit och fortsatte. Problemet som senare visade sig var att git på c9 skulle vara jobb och ville inte att jag skulle pusha mina commits vilket innebar att jag fick fula och lägga till css filen manuellt via github. Inte direkt optimalt, men bättre att ha en klar lösning än att gör om den. Anyway här är länken till laborationssidan. Copycat
